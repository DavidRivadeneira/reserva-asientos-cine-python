# Reserva de Asientos en Cine

Programa en Python que permite reservar un asiento en una sala de cine usando una matriz de 3 filas x 4 columnas.

## Funcionamiento
El usuario ingresa la fila y columna del asiento a reservar.  
El programa marca el asiento como reservado y muestra el estado de la sala.
